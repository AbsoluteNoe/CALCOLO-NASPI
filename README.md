# CALCOLO-NASPI
Uno script che ti permette di calcolare la fine dell'erogazione della tua NASpI

1- Aprire il link: https://www.programiz.com/python-programming/online-compiler/

2- Incollare il file "main.py" nella parte "main.py" del sito.
3- Clicca su RUN e segui le indicazioni.

N.B.: il formato data deve essere GG/MM/AAAA, utilizzre "/" come separatore
