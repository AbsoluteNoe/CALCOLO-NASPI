# CALCOLO-NASPI
Uno script che ti permette di calcolare la fine dell'erogazione della tua NASpI

1- Aprire in una nuova scheda il link: https://www.programiz.com/python-programming/online-compiler/
2- Cancellare tutto nella parte "main.py" del sito.
3- Aprire in una nuova scheda il link: https://raw.githubusercontent.com/AbsoluteNoe/CALCOLO-NASPI/refs/heads/main/main.py
4- Incollare il contenuto del link nella parte "main.py" del sito.
5- Clicca su RUN e segui le indicazioni.

N.B.: il formato data deve essere GG/MM/AAAA, utilizzre "/" come separatore
